Inventory Management System

This Inventory Management System offers a user-friendly interface, allowing users to efficiently manage their items. The system lets users easily add or remove items, ensuring optimal inventory management and organization.

Technical Highlights:

  • ViewModel - For managing UI-related data in a lifecycle-conscious way.
  • Room - Robust database library for fluid data access.
  • RecyclerView - For efficiently displaying large sets of data.
  • Fragments - Modular pieces of UI.
  • Navigation - For effective UI navigation between app destinations.
